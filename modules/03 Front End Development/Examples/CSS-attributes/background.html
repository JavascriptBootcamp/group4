CSS Backgrounds - Definition and usage

CSS background is an attribute which defines the background of elements. There are various kinds of background attributes like:

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position

Note: there are a few more background attributes like: background-clip,
background-blend-mode, background-origin et cetera.. in this description
we will be focused in the common kinds of the background attribute

background-color:

The background-color attirbute specify the background color of an element.
For example:
if we will use an embedded css style to color the background of a button:

button {
background-color: yellow;
}

graphic result will be like:

background-color:

The background-image attirbute enable an element to have an image background.
For example:
if we will use an embedded css style to color the background of a list items:

.blahblahItem {
background-image: url({Your image address};
}

graphic result will be like:

  • Blah blah number 1
  • Blah blah number 2
  • Blah blah number 3

background-repeat:

The background-repeat attirbute enable an background-image attribute to repeat itself vertically/horizontally.
For example:
if we will use an embedded css style to color the background of a list items:

.blahblahItem {
background-repeat: repeat-y; background-image: url({Your image address};
}

graphic result will be like:

  • Blah blah number 1
  • Blah blah number 2
  • Blah blah number 3

background-repeat: repeat-x will act respectively across the x axis.

background-attachment:

The background-attachment attirbute enable an background-image attribute to get sticky and wont scroll with the page(fixed).
For example:
if we will use an embedded css style to color the background of a list items:

.body {
background-attachment:fixed; background-repeat: no-repeat; background-image: url({Your image address};
}
