The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use tables floats positioning and inline-block.

Example 1

Set Columns in layout: grid-template-columns: auto auto;
Set Rows in layout: grid-template-rows: auto auto;

Example 4

The grid-area property can be used as a shorthand property for the grid-row-start, grid-column-start, grid-row-end and the grid-column-end properties.
grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end.
grid-area: 1 / 2 / 3 / 4;
